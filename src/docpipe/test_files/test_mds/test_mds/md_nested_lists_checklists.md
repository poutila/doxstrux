# Markdown Nested Lists and Checklists Test

This file is for testing **nested unordered lists, ordered lists, and checklists** in various combinations.

---

## 1. Nested Unordered Lists

- Level 1 - Item 1
  - Level 2 - Item 1.1
    - Level 3 - Item 1.1.1
    - Level 3 - Item 1.1.2
  - Level 2 - Item 1.2
- Level 1 - Item 2
  - Level 2 - Item 2.1

2. Step 2
